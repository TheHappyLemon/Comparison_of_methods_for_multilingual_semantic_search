Apache Tomcat (vai vienkārši Tomcat, iepriekš arī Jakarta Tomcat) ir atvērtā pirmkoda tīmekļa serveris un servletu konteiners, kuru izstrādā Apache Software Foundation (ASF). Tomcat realizē Sun Microsystems izstrādātās Java Servlet un JavaServer Pages (JSP) specifikācijas un piedāvā "tīru Java" HTTP tīmekļa servera vidi Java koda izpildei. Apache Tomcat iekļauj rīkus konfigurācijai un pārvaldībai, bet to var konfigurēt arī, labojot XML konfigurācijas failus.

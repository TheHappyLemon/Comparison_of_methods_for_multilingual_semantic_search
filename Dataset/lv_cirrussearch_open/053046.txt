Windows Home Server ir Microsoft izstrādātā un 2007. gada 7. janvārī paziņotā servera operētājsistēma, kas ir paredzēta mājas lietošanai un tās galvenais uzdevums ir apvienot mājokļos esošos datorus, to datus, veikt datu dublēšanu un nodrošināt attālināto piekļuvi. Windows Home Server operētājsistēma ir bāzēta uz Windows Server 2003 SP2 sistēmas. Plašam patēriņam Windows Home Server tika nodots 2007. gada 16.jūlijā.

Google Web Toolkit (GWT ˈɡwɪt) ir atvērtā koda programmēšanas rīku kopums, kurš ļauj tīmekļa izstrādātājiem radīt un uzturēt sarežģītas JavaScript lietojumprogrammas, izmantojot Java valodu. Neskaitot dažas platformai specifiskas bibliotēkas, viss pārējais ir Java kods, ko var kompilēt uz jebkuras no atbalstītajām platformām, izmantojot iekļautos GWT Ant kompilācijas failus. Tas ir pieejams, saskaņā ar Apache licences versiju 2.0. GWT piedāvā atkalizmantojamus, efektīvus risinājumus Ajax pielietošanai, to skaitā asinhroniem attālnātiem procedūru izsaukumiem, pārlūka vēstures pārvaldībai, grāmatzīmēm, internacionalizācijai un pārlūku savietojamībai.

Windows 9x ir vispārīgs nosaukums, kas attiecas uz Microsoft Windows datoru operētājsistēmu sēriju, kas tika radītas no 1995. līdz 2000. gadam un bija balstītas uz Windows 95 kodolu un MS-DOS pamatiem, kas nākamajās versijās tika uzlaboti. Šis apzīmējums iekļauj visas Windows 95, Windows 98 un Windows ME versijas. Windows 9x galvenokārt tika izmantots darbstacijām. 1998. gadā Windows piederēja 82% no operētājsistēmu tirgus. Windows 9x iekšējais versijas numurs bija 4.x. Iepriekšējās MS-DOS balstītās Windows versijas izmantoja versijas numuru 3.2 vai zemāku. Windows NT, kas bija paredzēts profesionāliem lietotājiem, tīkla datoriem un uzņēmumiem, izmantoja līdzīgu, bet atšķirīgu versijas numuru starp 3.1 un 4.0. Pēdējā Windows 9x sērijas operētājsistēma tika izlaista 2000. gada septembrī ar nosaukumu Windows ME, tās iekšējais versijas numurs bija 4.9. Visas turpmākās Windows versijas, sākot ar Windows 2000 un Windows XP, tika balstītas uz Windows NT izejas koda.

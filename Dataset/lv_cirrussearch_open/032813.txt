C standarta bibliotēka (angļu: C standard library) ir C programmēšanas valodas standarta bibliotēka, ko nosaka ANSI C standarts. Tas tika izstrādāts vienā laikā ar POSIX C bibliotēkas standartu, kas iekļauj vēl papildus funkcionalitāti. Tā kā ANSI C standartu vēlāk pieņēma arī Starptautiskā standartizācijas organizācija (ISO), to mēdz dēvēt arī par ISO C bibliotēku (angļu: ISO C Library). C standarta bibliotēka ietver makrodefinīcijas, datu tipu definīcijas un biežāk nepieciešamās funkcijas virkņu apstrādei, matemātiskiem aprēķiniem, ievades/izvades apstrādei, atmiņas pārvaldībai un piekļuvei dažādiem operētājsistēmas servisiem.

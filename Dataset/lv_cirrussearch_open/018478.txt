Windows Phone ir Microsoft mobilo operētājsistēmu sērija, Windows Mobile platformas pēctece, lai gan nav savietojama ar to. Atšķirībā no priekšteces, Windows Phone primāri ir mērķēts uz plaša patēriņā tirgu nevis uz uzņēmumiem. Pirmā versija Windows Phone 7 tika izlaista 2010. gada 21. oktobrī. 2014. gada 14. aprīlī tika izlaista Windows Phone 8.1, bet jaunākā versija ir Windows 10 Mobile. Operētājsistēmā Microsoft ieviesa jaunu lietotāja saskarni, kas veidota ar dizaina valodu Metro (Modern UI). Windows Phone 7 veidota uz Windows Mobile pamata, kas izmanto Windows CE kodolu. Windows Phone 8 pamatā ir Windows NT kodols ar daudziem komponentiem no Windows 8, tādējādi lietojumprogrammas ir viegli pārnesamas no vienas platformas uz otru. Windows Phone 7 izmantoja mobilo ierīču ražotāji Acer, Fujitsu, Nokia, LG, Samsung, HTC, Alcatel un ZTE. Windows Phone 8 izmantoja Nokia (Microsoft Mobile), Samsung, HTC, Huawei. Windows Phone 8 pieejama kopš 2012.gada 29. oktobra.

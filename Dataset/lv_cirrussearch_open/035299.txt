GitHub ir tīmeklī balstīts Git repozitoriju mitināšanas pakalpojums. Tajā tiek piedāvāta visa izplatītu versiju pārvaldības sistēmu un pirmkoda vadības funkcionalitāte, pieejamas ir arī vairākas citas iespējas, tostarp piekļuves vadība, kļūdu izsekošana, uzdevumu pārvaldība, wiki. GitHub repozitoriji var būt publiski pieejami vai privāti (maksas pakalpojums). Līdz 2016. gada aprīlim GitHub bija reģistrējušies 14 miljoni lietotāju, kopējais repozitoriju skaits pārsniedza 35 miljonus, tādējādi GitHub ir lielākā pirmkoda krātuve pasaulē. GitHub mājaslapa darbojas kopš 2008. gada aprīļa. Preču zīmes GitHub talismans ir Octocat, kas ir antropomorfs kaķis ar galvkāju locekļiem. 2018. gada 4. jūnijā uzņēmums Microsoft paziņoja par nodomu iegādāties GitHub par 7,5 miljardiem ASV dolāru.

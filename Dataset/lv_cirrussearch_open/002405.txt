Šis raksts ir par operētājsistēmu. Par citām jēdziena Vista nozīmēm skatīt nozīmju atdalīšanas lapu. Windows Vista ir Microsoft Windows saimes operētājsistēma personālajiem datoriem, kura pārdošanā tika izlaista 2007. gada 30. janvārī. Pirms nosaukuma "Vista" došanas 2005. gada 22. jūlijā tā bija pazīstama ar kodēto nosaukumu "Longhorn". Windows Vista tika izlaista vairāk nekā piecus gadus pēc tās priekšgājēja Windows XP, kas bija ilgākais pārtraukums starp Windows saimes operētājsistēmām. Windows Vista pēctece ir Windows 7, kas tika izlikta pārdošanā 2009. gada 22. oktobrī. Galvenais atbalsts priekš Windows Vista no Microsoft beidzās 2012. gada 10. aprīlī, un pagarinātais atbalsts beidzās 2017. gada 11. aprīlī.

Remote Desktop Protocol (RDP) ir Microsoft izstrādāts patentēts protokols, kas nodrošina lietotājam grafisku saskarni, lai izveidotu savienojumu ar citu datoru, izmantojot tīkla savienojumu. Lietotājs šim nolūkam izmanto RDP klienta programmatūru, bet otrā datorā ir jādarbina RDP servera programmatūra. Klienti pastāv lielākajai daļai Microsoft Windows versiju (tostarp Windows Mobile), Linux, Unix, macOS, iOS, Android un citām operētājsistēmām. RDP serveri ir iebūvēti Windows operētājsistēmās; pastāv arī RDP serveris Unix un OS X operētājsistēmām. Pēc noklusējuma serveris izmanto TCP 3389. portu un UDP 3389. portu. Microsoft pašlaik savu oficiālo RDP klienta programmatūru dēvē par Remote Desktop Connection, agrāk Terminal Services Client. Protokols ir ITU-T T.128 lietojumprogrammu koplietošanas protokola paplašinājums. Microsoft savā vietnē publiskojis dažas specifikācijas.

Kotlin ir atvērtā koda, stingri tipizēta programmēšanas valoda, kas atbalsta gan objektorientētu, gan funkcionālu programmēšanu. Valodas izstrādes izmaksas sedz kompānija JetBrains, savukārt Kotlin fonds aizsargā Kotlin preču zīmi. Valoda nosaukta Somu jūras līcī atrodošās Kotlinas salas, uz kuras atrodas Kronštate, vārdā. 2019. gada 7. maijā Google paziņoja, ka Kotlin programmēšanas valoda ir vēlamā valoda Android lietotņu izstrādātājiem. Kopš Android Studio 3.0 izlaišanas 2017. gada oktobrī Kotlin ir iekļauts kā alternatīva parastajam Java kompilatoram. Android Kotlin kompilators pēc noklusējuma ir vērsts uz Java 6, taču optimizēšanai ļauj programmētājam izvēlēties starp Java 8 līdz Java 13.

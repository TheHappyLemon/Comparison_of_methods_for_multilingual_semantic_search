React ir JavaScript bibliotēka lietotāja saskarņu veidošanai. Bibliotēku uztur Facebook un individuālu izstrādātāju un uzņēmumu kopiena. Uz React bāzes var izstrādāt vienas lapas tīmekļa lietotnes (SPA) vai arī mobilās lietotnes. Taču React atbild vienīgi par datu attēlošanu, tāpēc pilnvērtīgām lietotnēm nepieciešamas citas bibliotēkas lietotnes stāvokļa pārvaldīšanai, maršrutēšanai, mijiedarbībai ar API. React izveidoja Facebook programmētājs Džordans Volks. Tā pirmoreiz Facebook tika izmantota 2011. gadā jaunumu plūsmā, 2012. gadā tā tika ieviesta Instagram. 2015. gadā tika izziņots React Native, ar kura palīdzību var veidot lietotnes Android, iOS vidē ar React palīdzību.

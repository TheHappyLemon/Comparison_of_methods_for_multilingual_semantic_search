'''''PHP: Hypertext Preprocessor''''' (no {{val|en| }} "PHP Hiperteksta priekšprocesors") ir [[atvērtais pirmkods|atvērtā pirmkoda]] [[skriptu valoda]], kura sākotnēji bija paredzēta [[serveris|servera]] puses lietojumos [[dinamiska tīmekļa lapa|dinamiska tīmekļa lapu]] ģenerēšanai. PHP var izmantot arī ar [[komandrindas saskarne|komandrindas saskarni]].

PHP ir no platformām neatkarīga valoda, kas nozīmē, ka kodēt programmu var vienā vidē, bet izplatīt citā vidē. PHP var darbināt gan ''Windows'' operētājsistēmā, gan uz daudzām ''Unix'' versijām un ''Mac OS X''.

PHP ir [[brīvā programmatūra]], un tiek izplatīta ar [[PHP licence]]s noteikumiem.

== Vēsture ==
1994. gadā dāņu programmētājs [[Rasmuss Lerdorfs]] izstrādāja nelielu ''[[Perl]]'' skriptu kopumu savai tīmekļa vietnei, kuru nosauca '''''Personal Home Page Tools'''''. Taču drīz vien ar ''Perl'' skriptiem vien nepietika, un tie tika pārrakstīti [[C (programmēšanas valoda)|C]] valodā un papildināti ar jaunām iespējām.

1997. gadā divi Izraēlas Tehnoloģiju institūta zinātnieki pārrakstīja PHP parseri, kas bija pamats PHP 3. 1998. gadā viņi pārrakstīja arī PHP pamata funkcijas, radot [[Zend dzinējs|Zend dzinēju]] (''Zend Engine'').

2000. gada maijā iznāca PHP 4, kas saturēja ''Zend Engine 1.0''.

2004. gada 13. jūlijā iznāca PHP 5 ar ''Zend Engine II''.

2005. gadā sākās darbs pie PHP 6. versijas izstrādes. Tajā viens no svarīgākajiem uzlabojumiem bija paredzēts pilnīgs [[unikods|unikoda]] atbalsts. Tomēr tā implementēšana izrādījās pārāk sarežģīta, kā arī radīja ātrdarbības problēmas, tāpēc galu galā unikoda atbalsta izmaiņas tika atmestas, bet pārējās sagatavotās izmaiņas ienestas PHP ar versijām 5.3 un 5.4.

2015. gada nogalē iznāca PHP 7. versija ar ''Zend Engine 3'', bet 6. versijas numurs tika izlaists, lai sākotnēji ieplānotās (un jau iespieddarbos daļēji aprakstītās) PHP 6 iespējas nejuktu ar reāli īstenotajām.

2020. gada 26. novembrī iznāca PHP 8, kas šobrīd ir visplašāk izmantotā PHP versija. PHP 8 ir izmaiņas, kas nav saderīgas ar iepriekšējām versijām.<ref name="PHP-8-Izlaists">{{Tīmekļa atsauce |title=PHP 8 Released|url=https://www.php.net/releases/8.0/en.php |website=PHP}}</ref><ref name="PHP-8-Statistika">{{Tīmekļa atsauce |title=PHP 8 versiju statistika 2023 |url=https://stitcher.io/blog/php-version-stats-july-2023}}</ref>

== Sintakse ==
<!--
<div style="float:right; margin:0 0 0 1em">
<source lang="php">
<html>
  <head>
    <title>PHP Test</title>
  </head>
  <body>
  <?php
  echo 'Sveiciens pasaulei!';
  ?>
  </body>
</html>
</source><br />''PHP code embedded within [[HTML]] code''</div>
-->
PHP parsē tam nodoto tekstu tikai starp norobežotājiem. Ārpus tiem teksts tiek izvadīts tieši, bez apstrādes. Norobežotāji var būt <code><?php</code>, lai sāktu, un <code>?></code>, lai noslēgtu PHP sekciju. Var būt arī šādi norobežotāji: <code><script language="php"></code> un <code></script></code>. Var izmantot arī norobežotāju saīsinātās formas <code><?</code> un <code>?></code>, kā arī [[Active Server Pages|ASP]] stila norobežotājus <code><%</code> un <code>%></code>.

== Izmantošana ==
Pamatā PHP izmanto uz [[Tīmekļa serveris|tīmekļa servera]] kā skriptu valodu, ar kuru sagatavo [[Tīmekļa lappuse|tīmekļa lapas]]. To bieži izmanto arī kā komandrindu [[Skriptu valoda|skriptu valodu]].
Visbiežāk PHP izmanto ar ''Linux'' serveriem kopā ar [[Apache HTTP serveris|''Apache'']] vai ''[[Nginx]]'' [[Tīmekļa serveris|HTTP serveri]] un ''[[MySQL]]'', ''[[PostgreSQL]]'' vai ''[[MariaDB]]'' datu bāzi.

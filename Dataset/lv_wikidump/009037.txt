{{Elektrodinamika}}
'''Elektriskais lādiņš''' ir [[fizikāls lielums]], kas nosaka [[Elektromagnētisms|elektromagnētisko]] mijiedarbību intensitāti <ref>{{Tīmekļa atsauce|url=https://www.siic.lu.lv/fiz/IT/F_12/info_fiz/e.html|title=elektriskais lādiņš|website=siic.lu.lv|access-date=17.08.2021}}</ref>, [[Fizika|fizikā]] to apzīmē ar <math>q \ </math> un tā [[mērvienība]] ir [[kulons]] (C).

Elektriskais lādiņš var būt pozitīvs vai negatīvs.

Lādiņiem un uzlādētiem ķermeņiem ir spēkā [[elektriskā lādiņa nezūdamības likums]] un [[Kulona likums]].

== Elementārlādiņš ==

Vismazākais (pēc [[Absolūtā vērtība|absolūtās vērtības]]) lādiņš piemīt [[Elektrons|elektronam]]. [[Eksperiments|Eksperimentāli]] ir konstatēts, ka tā lādiņš <math>e</math> ir −1,6·10<sup>−19</sup> C (tāds pats pēc absolūtās vērtības lādiņš piemīt arī [[Protons|protonam]], tikai tas ir pozitīvs), to sauc par elementārlādiņu. Jebkurš cits lādiņš ir šī lādiņa daudzkārtnis. <ref name=":0">{{Grāmatas atsauce|title=Fizikas rokasgrāmata|last=Fļorovs|first=V.|publisher=Zvaigzne|year=1985|pages=157.—158|last2=Kolangs|first2=I.|last3=Puķītis|first3=P.|last4=Šilters|first4=E.}}</ref>

=== Elektrona lādiņš un Faradeja skaitlis ===
[[Fizikāla konstante|Konstanti]] <math>F=eN_A</math>, kur <math>N_A</math> ir [[Avogadro skaitlis]], sauc par [[Faradeja skaitlis|Faradeja skaitli]], tā vērtība ir 96 484,56 [[Kulons|C]]/[[Mols|mol]]. Pēc [[Faradeja likumi elektrolīzei|Faradeja likumiem elektrolīzei]], <math>F=\frac{C}{m}q</math>  un Faradeja skaitlis ir elektriskais lādiņš <math>q</math>, ko [[Elektrolīts|elektrolītā]] pārnes [[Jons|joni]], kuru kopējā [[masa]] <math>m</math> skaitliski vienāda ar vielas [[Ķīmiskais ekvivalents|ķīmisko ekvivalentu]] <math>C</math>, tātād, nosakot eksperimentāli <math>m</math> un <math>q</math>, aprēķina Faradeja skaitli un, zinot to, aprēķina [[Elektrons|elektrona]] lādiņu. <ref name=":02">{{Grāmatas atsauce|title=Fizikas rokasgrāmata|last=Fļorovs|first=V.|publisher=Zvaigzne|year=1985|pages=227.—228|last2=Kolangs|first2=I.|last3=Puķītis|first3=P.|last4=Šilters|first4=E.}}</ref>

== Elektrizācija ==
Elektrizācija ir elektrisko lādiņu atdalīšanās un to uzkrāšanās [[Fizikāls ķermenis|ķermenī]] <ref>{{Tīmekļa atsauce|url=https://samelectrik.ru/chto-takoe-elektrizaciya-tel.html|title=Что такое электризация тел и как она происходит|website=samelectrik.ru|access-date=17.08.2021}}</ref>, šī procesa rezultātā ķermenis uzlādējas pozitīvi vai negatīvi — ja ķermenis iegūst pozitīvus vai negatīvus lādiņnesējus (daļiņas), rodas attiecīgās zīmes elektriskā lādiņa pārpalikums; ja ķermenis lādiņnesējus zaudē, rodas attiecīgās zīmes elektriskā lādiņa iztrūkums.

Parasti (bez elektrizācijas) [[viela]] ir elektroneitrāla, tas ir, pretēju zīmju lādiņi tajā savstarpēji kompenesējas ([[atoms]], kurš sastāv no pozitīvi lādēta [[Atoma kodols|kodola]] un negatīvi lādēta elektronu apvalka, ir elektroneitrāls). <ref name=":0" /> 

== Tilpuma lādiņa blīvums ==

Tilpuma lādiņa blīvumu [[fizika|fizikā]] apzīmē ar <math>\rho \ </math>

: <math>\rho = \lim_{\Delta V \rarr 0} \frac{\Delta q}{\Delta V} \ </math>
::: kur
:::: <math>\Delta V \ </math> - [[tilpums|tilpuma]] elements
:::: <math>\Delta q \ </math> - [[lādiņš]], kurš atrodas dotajā tilpumā

Saskaņā ar šo [[definīcija|definīciju]] lādiņš ir vienāds ar [[integrālis|integrāli]]

:<math>q = \int_V \rho \mathrm{d} V \ </math>

== Virsmas lādiņa blīvums ==

Virsmas lādiņa blīvumu apzīmē ar <math>\rho_S \ </math>

: <math>\rho_S = \lim_{\Delta S \rarr 0} \frac{\Delta q}{\Delta S} \ </math>
::: kur
:::: <math>\Delta S \ </math> - [[virsma]]s elements
:::: <math>\Delta q \ </math> - [[lādiņš]], kurš atrodas uz dotās virsmas

Saskaņā ar šo [[definīcija|definīciju]] lādiņš ir vienāds ar [[integrālis|integrāli]]

:<math>q = \int_S \rho_S \mathrm{d} S \ </math>

== Lineārais lādiņa blīvums ==

Lineāro lādiņa blīvumu apzīmē ar <math>\rho_l \ </math>

:<math>\rho_l = \lim_{\Delta l \rarr 0} \frac{\Delta q}{\Delta l} \ </math>
::: kur
:::: <math>\Delta l \ </math> - [[līnija]]s elements
:::: <math>\Delta q \ </math> - [[lādiņš]], kurš atrodas uz dotās virsmas

Saskaņā ar šo [[definīcija|definīciju]] lādiņš ir vienāds ar [[integrālis|integrāli]]

:<math>q = \int_l \rho_l \mathrm{d} l \ </math>

== Delta funkcija ==

Pieņemsim, ka uz <math>x \ </math> ass punktā <math>x_0 \ </math> atrodas [[punktveida lādiņš]] <math>q \ </math>. Visos <math>x \ </math> ass [[punkts|punktos]] [[lādiņa blīvums]] <math>\rho (x) = 0\ </math>, izņemot punktu <math>x = x_0 \ </math>, kurā tas ir bezgalīgi liels, jo punktam nav [[tilpums|tilpuma]].

Lai gan [[funkcija]] <math>\rho (x) \ </math> nav nepārtraukta, to var izteikt [[matemātika|matemātiski]] šādi:

: <math>\rho (x) = q \delta (x - x_0) \ </math>
::: kur
:::: <math>\delta (x - x_0) = \begin{cases} \ 0, & x \ne x_0 \\ \infty, & x = x_0 \end{cases} \ </math> ([[Delta funkcija]])

Vēl jābūt izpildītam šādam nosacījumam:

: <math>\int_{-\infty}^{+\infty} \delta (x - x_0) \mathrm{d} x = 1 \ </math>,

kurš nepieciešams, lai iegūtu galīgu lielumu <math>q \ </math>.

: <math>\int_{-\infty}^{+\infty} \rho (x) \mathrm{d} x = q \int_{-\infty}^{+\infty} \delta (x - x_0) \mathrm{d} x = q \ </math>

=== Vairāku lādiņu blīvums ===

Situācija ir līdzīga, ja uz <math>x \ </math> ass diskrētos punktos <math>x_i \ </math> izvietoti <math>n \ </math> [[punktveida lādiņš|punktveida lādiņi]] <math>q_i \ </math> un [[sistēma]]s pilnais [[lādiņš]] ir <math>q = \sum_{i=1}^n q_i \ </math>. Arī šādu lādiņu izvietojumu var izteikt ar <math>\delta \ </math> funkcijām <math>\delta (x - x_i) \ </math>.

: <math>\rho (x) = \sum_{i=1}^n q_i \delta (x - x_i) \ </math>

Un līdz ar to

: <math>\int_{-\infty}^{+\infty} \rho (x) \mathrm{d} x = \sum_{i=1}^n q_i \int_{-\infty}^{+\infty} \delta (x - x_i) \mathrm{d} x = \sum_{i=1}^n q_i = q \ </math>.

== Punktveida elektriskais lādiņš ==
Par punktveida lādiņiem var uzskatīt elektriski lādētus ķermeņus, attālums starp kuriem ir ievērojami lielāks par šo ķermeņu izmēriem. <ref name=":0" />

== Vēsture ==
Jau [[Senā Grieķija|senie grieķi]] iepazinuši saberzētam [[Dzintars|dzintaram]] piemītošo spēju pievilkt vieglus priekšmetus. [[Renesanse|Renesansē]] sākti sistemātiski elektrisko parādību pētījumi, 18. gadsimtā atrastas pirmās kvalitatīvās likumsakarības. [[Bendžamins Franklins]] (18. gs.) atklāja abu veidu elektriskos lādiņus. 19. gs. 30. gados [[Maikls Faradejs]] [[Eksperiments|eksperimentāli]] pierādīja elektrisko lādiņu nezūdamību. 1911. gadā [[Roberts Milikens]] savā eksperimentā secināja, ka elektrona lādiņa modulis ir elektriskā lādiņa mazākā porcija. <ref name=":0" />

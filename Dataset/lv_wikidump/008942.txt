'''Magnētiskā lauka indukcija''' jeb '''magnētiskā indukcija''' ir [[fizikāls lielums]], kas raksturo [[magnētiskais lauks|magnētiskā lauka]] [[induktivitāte|induktivitāti]].

Magnētiskā lauka indukciju [[fizika|fizikā]] apzīmē ar <math>B \ </math>un tās [[Starptautiskā mērvienību sistēma|SI]] [[mērvienība]] ir [[tesla]] (T).

== Taisna strāvas vada magnētiskā indukcija ==

[[Elektriskā strāva|Strāvas]] magnētiskā lauka indukciju aprēķina pēc [[Bio—Savāra—Laplasa likums|Bio—Savāra—Laplasa likuma]].

=== Skalārā forma ===

Ja [[elektriskā strāva|strāva]] <math>I \ </math> plūst pa bezgalīgi garu, taisnu un tievu vadu, tad strāvas magnētiskais lauks <math>B \ </math> attālumā <math>r \ </math> no vada pēc [[Bio — Savāra — Laplasa likums|Bio — Savāra — Laplasa likuma]] ir
: <math>B = k_M \frac{I}{r} \ </math>
::: kur
:::: <math>k_M \ </math> - koeficients (2×10<sup>-7</sup> [[henrijs|H]]/[[metrs|m]])
:::: <math>I \ </math> - [[elektriskā strāva|strāvas stiprums]] ([[ampērs|A]])

==== Magnētiskā konstante ====
[[Attēls:Magnetiskais lauks.JPG|thumb|250px]]

Koeficients <math>k_M \ </math> ir izsakāms šādi:
: <math>k_M = \frac{\mu_0}{2\pi} \ </math>
::: kur
:::: <math>\mu_0 \ </math> - [[magnētiskā konstante]] (1.256637×10<sup>-6</sup> [[henrijs|H]]/[[metrs|m]])
:::: <math>\pi \ </math> - 3.141593

Ievērojot to, taisna strāvas vada magnētiskā lauka indukciju <math>B \ </math> aprēķina pēc formulas
: <math>B = \mu_0 \frac{I}{2\pi r} \ </math>

=== Vektoriālā forma ===

: <math>\vec{B} = \mu_0 \frac{I}{2\pi r} \vec{\tau} \ </math>
::: kur
:::: <math>\vec{\tau} \ </math> - pieskares vektors

== Lorenca spēks ==

: <math>\vec{B} = \frac{1}{q v_T^2} \vec{F}_L \times \vec{v}_T \ </math>
::: kur
:::: <math>q \ </math> - lādiņš, uz kuru darbojas Lorenca spēks ([[kulons|C]]);
:::: <math>\vec{v}_T \ </math> - magnētiskajai indukcijai perpendikulārā ātruma komponente;
:::: <math>\vec{F}_L \ </math> - Lorenca spēka vektors ([[ņūtons|N]]);

''Galvenais raksts'': [[Lorenca spēks]]

== Magnētiskās indukcijas cirkulācija ==

Magnētiskās indukcijas cirkulāciju [[fizika|fizikā]] apzīmē ar <math>\oint_l \vec{B} \mathrm{d}\vec{r} \ </math>.

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \mu_0 I \ </math>
::: kur
:::: <math>\mu_0 \ </math> - magnētiskā konstante (1.256637×10<sup>-6</sup> [[henrijs|H]]/[[metrs|m]])
:::: <math>I \ </math> - [[elektriskā strāva|strāvas stiprums]] vadā, ap kuru ir apvilkts kontūrs <math>l \ </math> ([[ampērs|A]])

''Galvenais raksts'': [[Magnētiskās indukcijas cirkulācija]]

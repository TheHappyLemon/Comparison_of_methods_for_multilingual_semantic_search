'''Delta funkciju''' (<math>\delta \ </math> funkciju) [[fizika|fizikā]] izmanto, lai raksturotu [[diskrēts|diskrētu]] [[punktveida lādiņš|punktveida lādiņu]] izvietojumu [[telpa|telpā]]. Šo [[funkcija|funkciju]] teorētiskajā fizikā pirmais sāka izmantot ievērojamais [[Anglija|angļu]] fiziķis [[Pols Adriens Moriss Diraks]], viens no [[kvantu mehānika]]s pamatlicējiem. Tādēļ delta funkciju dažreiz sauc arī par ''Diraka delta funkciju''. Tā pieder pie vispārinātajām [[funkcija|funkcijām]]; tā nav nepārtraukta un diferencējama šo jēdzienu parastajā nozīmē. Tā ir ērta, ja, piemēram, jāaprēķina [[materiāls punkts|materiālo punktu]] sistēmu raksturojošie [[integrāls lielums|integrālie lielumi]] (pilnais [[lādiņš]], [[elektriskā lauka intensitāte]] utt.).

== Lādiņa blīvums ==

[[Attēls:Dirac distribution PDF.png|300px|thumb|left|Shematiski parādīta delta funkcija (līnija ar bultiņu galā). Bultas augstums parāda funkcijas vērtību.]]

Pieņemsim, ka uz <math>x \ </math> ass punktā <math>x_0 \ </math> atrodas [[punktveida lādiņš]] <math>q \ </math>. Visos <math>x \ </math> ass [[punkts|punktos]] [[lādiņa blīvums]] <math>\rho (x) = 0\ </math>, izņemot punktu <math>x = x_0 \ </math>, kurā tas ir bezgalīgi liels, jo punktam nav [[tilpums|tilpuma]].

Lai gan [[funkcija]] <math>\rho (x) \ </math> nav nepārtraukta, to var izteikt [[matemātika|matemātiski]] šādi:

: <math>\rho (x) = q \delta (x - x_0) \ </math>
::: kur
:::: <math>\delta (x - x_0) = \begin{cases} \ 0, & x \ne x_0 \\ \infty, & x = x_0 \end{cases} \ </math>

Vēl jāizpildās šādam nosacījumam:

: <math>\int_{-\infty}^{+\infty} \delta (x - x_0) \mathrm{d} x = 1 \ </math>,

kurš nepieciešams, lai iegūtu galīgu lielumu <math>q \ </math>.

: <math>\int_{-\infty}^{+\infty} \rho (x) \mathrm{d} x = q \int_{-\infty}^{+\infty} \delta (x - x_0) \mathrm{d} x = q \ </math>

=== Vairāku lādiņu blīvums ===

Situācija ir līdzīga, ja uz <math>x \ </math> ass diskrētos punktos <math>x_i \ </math> izvietoti <math>n \ </math> [[punktveida lādiņš|punktveida lādiņi]] <math>q_i \ </math> un [[sistēma]]s pilnais [[lādiņš]] ir <math>q = \sum_{i=1}^n q_i \ </math>. Arī šādu lādiņu izvietojumu var izteikt ar <math>\delta \ </math> funkcijām <math>\delta (x - x_i) \ </math>.

: <math>\rho (x) = \sum_{i=1}^n q_i \delta (x - x_i) \ </math>

Un līdz ar to

: <math>\int_{-\infty}^{+\infty} \rho (x) \mathrm{d} x = \sum_{i=1}^n q_i \int_{-\infty}^{+\infty} \delta (x - x_i) \mathrm{d} x = \sum_{i=1}^n q_i = q \ </math>

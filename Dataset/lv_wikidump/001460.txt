There are 2 fields that can be used with this template:
# The name of the file on Commons
# If the local file should NOT be deleted you can add a reason. For example because of {{tl|FoP-Latvia}}.

Below are some examples on how to use the template:

;File have the same name on Commons and local file can be deleted
<pre>{{NowCommons}}</pre>

;File have a different name on Commons and local file can be deleted
<pre>{{NowCommons|Example.svg}}</pre>

;File have the same name on Commons and local should NOT be deleted
<pre>{{NowCommons|2=Some reason to keep the local file}}</pre>

;File have a different name on Commons and local file should NOT be deleted
<pre>{{NowCommons|Example.svg|Some reason to keep the local file}}</pre>

The template will add the files to one of these categories:
* [[:Kategorija:NowCommons]] (if the file have a different name on Commons)
* [[:Kategorija:Fails ir Vikikrātuvē ar tādu pašu nosaukumu]] (if the file have the same name on Commons)
* [[:Kategorija:Fails ir Vikikrātuvē ar tādu - nepavisam]] (if the local file should NOT be deleted)

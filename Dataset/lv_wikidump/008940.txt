{{Elektrodinamika}}
'''Magnētiskās indukcijas cirkulāciju''' [[fizika|fizikā]] apzīmē ar <math>\oint_l \vec{B} \mathrm{d}\vec{r} \ </math>.

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \mu_0 (I + I_D) \ </math>
::: kur
:::: <math>\mu_0 \ </math> - [[magnētiskā konstante]] (1.256637×10<sup>-6</sup> [[henrijs|H]]/[[metrs|m]])
:::: <math>I \ </math> - [[elektriskā strāva|strāvas stiprums]] vadā, ap kuru ir apvilkts kontūrs <math>l \ </math> ([[ampērs|A]])
:::: <math>I_D \ </math> - [[nobīdes strāva]] ([[ampērs|A]]) 

Ja [[elektriskais lauks]] laikā nemainas, tad:

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \mu_0 I \ </math>

== Magnētiskās indukcijas cirkulācijas pierādījums ==

: <math>\vec{B} \mathrm{d}\vec{r} = B \mathrm{d} r \cos\alpha \ </math>

Līdz ar to, cirkulācija ir:

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \oint_l B \mathrm{d} r \cos\alpha \ </math>
::: kur
:::: <math>B \ </math> - [[magnētiskais lauks|magnētiskā lauka]] [[indukcija]] ([[tesla|T]])
:::: <math>\mathrm{d} r = r_0 - r\ </math>
:::: <math>\alpha \ </math> - [[leņķis]] starp <math>\mathrm{d} \vec{r} \ </math> un <math>\vec{B} \ </math> ([[radiāns|rad]])
[[Attēls:Magnetiska lauka cirkulacija.JPG|450px|thumb|right]]

Savukārt 

: <math>\mathrm{d} r = r \mathrm{d}\varphi \ </math>
::: kur
:::: <math>r \ </math> - [[attālums]] no [[elektriskā strāva|strāvas]] vada 
:::: <math>\varphi \ </math> - leņķis starp diviem [[rādiusvektors|rādiusvektoriem]] <math>\vec{r} \ </math> un <math>\vec{r}_0 \ </math>

Tādēļ

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \oint_l B r \mathrm{d}\varphi \cos\alpha = B r \oint_l \mathrm{d}\varphi \cos\alpha \ </math>
: <math>\oint_l \mathrm{d}\varphi \cos\alpha = 2 \pi \ </math>
: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = 2 \pi B r \ </math>

Lai cirkulāciju pabeigtu aprēķināt pa noslēgtu [[kontūrs|kontūru]] <math>l \ </math>, kurš aptver [[elektriskā strāva|strāvu]] <math>I \ </math> tai [[perpendikulārs|perpendikulārā]] [[plakne|plaknē]], izmanto [[Bio-Savāra-Laplasa likums|Bio-Savāra-Laplasa likumu]].

: <math>B = \mu_0 \frac{I}{2 \pi r} \ </math>

No tā izriet, ka

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \frac{2 \pi \mu_0 r I}{2 \pi r} = \mu_0 I \ </math>

=== Nobīdes strāva ===

Jāņem vērā vēl [[nobīdes strāva]] <math>I_D \ </math>, kuru rada mainīga [[elektriskais lauks|elektriskā lauka]] intensitātes [[Gausa teorēma|plūsma]]. Tādā gadījumā

: <math>\oint_l \vec{B} \mathrm{d}\vec{r} = \mu_0 (I + I_D) \ </math>

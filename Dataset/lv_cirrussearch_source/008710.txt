Git ir sadalīta versiju kontroles sistēma ar uzsvaru uz ātrdarbību. Git sākotnēji izplānoja un izstrādāja Līnuss Tūrvaldss Linux kodola izstrādei. Katra Git jeb repozitorijs ietver pilnu izmaiņu vēsturi un nodrošina vadības iespējas, neatkarīgi no tīkla savienojuma pieejamības vai centrālā servera. Šībrīža Git uzturēšanu uzrauga Džunio Hamano.

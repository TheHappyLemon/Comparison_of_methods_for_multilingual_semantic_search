X Window System ir logošanas sistēma, kas nodrošina standarta instrumentus un protokolus lietotāja grafisko saskarņu izveidei. Gandrīz visas mūsdienīgās operētājsistēmas atbalsta X Window System, tomēr plašāko izmantojumu tā ir radusi tieši UNIX tipa sistēmās. X Window System nodrošina grafiskās saskarnes pamatfunkcijas: logu attēlošanu un pārvietošanu, mijiedarbību ar peli un tastatūru. X Window System nenosaka lietotāja grafiskās saskarnes detaļas, par to atbild kāds no daudzskaitlīgajiem logu pārvaldniekiem. Šī iemesla dēļ programmu grafiskais atainojums X Window System vidē var ievērojami atšķirties; dažādas programmas var izmantot krasi atšķirīgas saskarnes. X Window System ir paredzēta tīkla caurspīdības (network transparency) funkcija: grafiskās programmas var tikt darbinātas uz citas darbstacijas tīklā, bet to saskarnes tiks pārraidītas pa tīklu un attēlotas uz lietotāja lokālās stacijas. X Window System gadījumā termini "klients" un "serveris" lielai daļai lietotāju rada neizpratni: "serveris" ir lietotāja lokālais displejs (displeja serveris), savukārt "klients" - programma, kas šo displeju izmanto un tā var tikt darbināta uz attālināta datora. X Window System izstrādāja Masačūsetsas Tehnoloģiju institūtā (MIT) 1984. gadā. Pašreizējā - X11 - protokola versija dienasgaismu ieraudzīja 1987. gadā. X projektu koordinē X.Org fonds. Atsaucēs bāzētā sistēmas implementācija (reference implementation), X.Org server, ir brīvi pieejams un tās izplatīšanu nosaka MIT licence un tai līdzīgu licenču noteikumi. Liela daļa modernai logošanas sistēmai piemītošu iespēju kā, piemēram, daudzmonitoru atbalsts (X Windows vidē - Xinerama), 3D efekti darbvirsmai (AIGLX), skārienjūtīga ekrāna funkcijas MPX, "iksos" bija nodrošinātas vēl pirms Windows piedāvāja šādu iespēju.

Lopitāla kārtula ļauj netieši atrast funkciju dalījuma robežu, ja tiešie robežas noteikšanas paņēmieni noved līdz nenoteiktībai 0 0 {\displaystyle {\tfrac {0}{0}}} vai ± ∞ ± ∞ {\displaystyle {\tfrac {\pm \infty }{\pm \infty }}} . Šim nolūkam sākotnējo funkciju dalījuma vietā aplūko to atvasinājumu dalījumu, kas daudzos gadījumos palīdz atbrīvoties no nenoteiktības. Kārtula nosaukta franču matemātiķa Gijoma Lopitāla (1661–1704) vārdā, kaut arī to atklāja Lopitāla skolotājs, Šveices matemātiķis Johans (I) Bernulli. Lopitāls bija pirmais, kurš šo kārtulu nopublicēja — savā diferenciālrēķinu mācības grāmatā 1696. gadā. Ja eksistē robeža lim x → c f ′ ( x ) g ′ ( x ) = L {\displaystyle \lim _{x\to c}{\frac {f'(x)}{g'(x)}}=L} , tad lim x → c f ( x ) g ( x ) = lim x → c f ′ ( x ) g ′ ( x ) = L {\displaystyle \lim _{x\to c}{\frac {f(x)}{g(x)}}=\lim _{x\to c}{\frac {f'(x)}{g'(x)}}=L} . Noteikumi: c {\displaystyle c} un L {\displaystyle L} ir skaitļi no paplašinātās reālo skaitļu ass (t. i. ass, kas iekļauj ne tikai reālos skaitļus, bet arī negatīvo bezgalību − ∞ {\displaystyle -\infty } un pozitīvo bezgalību + ∞ {\displaystyle +\infty } ). Funkcijām f ( x ) {\displaystyle f(x)} un g ( x ) {\displaystyle g(x)} jābūt diferencējamām kādā atvērtā intervālā I {\displaystyle I} , kurš ietver sevī punktu c {\displaystyle c} vai kuram punkts c {\displaystyle c} ir viens no galiem. Līdz ar to aplūkojamā robeža var būt arī vienpusīga ( lim x → c + f ( x ) g ( x )  vai  lim x → c − f ( x ) g ( x ) ) {\displaystyle \left(\lim _{x\to c^{+}}{\frac {f(x)}{g(x)}}{\text{ vai }}\lim _{x\to c^{-}}{\frac {f(x)}{g(x)}}\right)} . g ′ ( x ) ≠ 0 {\displaystyle g'(x)\neq 0} visā intervālā I {\displaystyle I} (izņemot, varbūt, punktu c {\displaystyle c} , ja tas atrodas intervāla iekšienē un aplūkojamajā uzdevumā ir vienāds nullei). Apgrieztais princips nedarbojas: no tā, ka robežvērtība lim f ( x ) g ( x ) {\displaystyle \textstyle \lim {\tfrac {f(x)}{g(x)}}} eksistē, nevar izsecināt, ka eksistē arī lim f ′ ( x ) g ′ ( x ) {\displaystyle \textstyle \lim {\tfrac {f'(x)}{g'(x)}}} . Tāpēc aplūkosim lim x → π / 2 f ′ ( x ) g ′ ( x ) {\displaystyle \lim _{x\to \pi /2}{\tfrac {f'(x)}{g'(x)}}} . Ja šī robeža eksistē, varēs izmantot Lopitāla kārtulu. lim x → π / 2 f ′ ( x ) g ′ ( x ) = lim x → π / 2 cos ⁡ 2 x sin ⁡ x − e cos ⁡ x = lim x → π / 2 − 2 sin ⁡ 2 x cos ⁡ x + e cos ⁡ x sin ⁡ x = ( − 2 ) ⋅ ( − 1 ) 0 + 1 ⋅ 1 = 2 {\displaystyle \lim _{x\to \pi /2}{\frac {f'(x)}{g'(x)}}=\lim _{x\to \pi /2}{\frac {\cos 2x}{\sin x-e^{\cos x}}}=\lim _{x\to \pi /2}{\frac {-2\sin 2x}{\cos x+e^{\cos x}\sin x}}={\frac {(-2)\cdot (-1)}{0+1\cdot 1}}=2} . Lopitāla kārtulas izpratnē ∞ {\displaystyle \infty } ir skaitlis un robežvērtība ∞ {\displaystyle \infty } tiek uzskatīta par eksistējošu. Tātad, saskaņā ar Lopitāla kārtulu, lim x → ∞ x ln ⁡ ( x ) = ∞ {\displaystyle \lim _{x\to \infty }{\frac {\sqrt {x}}{\ln(x)}}=\infty } .   Ja pēc vienas diferencēšanas atkal rodas nenoteiktība 0 0 {\displaystyle {\tfrac {0}{0}}} vai ± ∞ ± ∞ {\displaystyle {\tfrac {\pm \infty }{\pm \infty }}} , Lopitāla kārtulu var pielietot atkārtoti, aplūkojot otro atvasinājumu attiecību, utt. Aplūkosim sekojošo piemēru, kurā Lopitāla kārtula tiek uzreiz pielietota divas reizes ātras risināšanas labad. lim x → 1 f ( x ) g ( x ) = lim x → 1 2 x 2 − x − 1 x 2 − x = lim x → 1 f ′ ( x ) g ′ ( x ) = lim x → 1 4 x − 1 2 x − 1 = lim x → 1 f ″ ( x ) g ″ ( x ) = lim x → 1 4 2 = 2 {\displaystyle \lim _{x\to 1}{\frac {f(x)}{g(x)}}=\lim _{x\to 1}{\frac {2x^{2}-x-1}{x^{2}-x}}=\lim _{x\to 1}{\frac {f'(x)}{g'(x)}}=\lim _{x\to 1}{\frac {4x-1}{2x-1}}=\lim _{x\to 1}{\frac {f''(x)}{g''(x)}}=\lim _{x\to 1}{\frac {4}{2}}=2} Šis rezultāts ir nepareizs. Kļūda ir radusies tāpēc, ka pirms otrās diferencēšanas netika ievērota prasība par nenoteiktības esamību. Pirmie atvasinājumi dod noteiktu dalījumu 4 ⋅ 1 − 1 2 ⋅ 1 − 1 {\displaystyle {\tfrac {4\cdot 1-1}{2\cdot 1-1}}} , no kura var tieši atrast pareizo robežvērtību 3 {\displaystyle 3} .   Prasība par robežvērtības lim x → c f ′ ( x ) g ′ ( x ) {\displaystyle \lim _{x\to c}{\frac {f'(x)}{g'(x)}}} esamību ir būtiska, jo dažas funkcijas f ′ {\displaystyle f\,'} un/vai g ′ {\displaystyle g\,'} var nerimstoši svārstīties. Ja tas notiek, Lopitāla kārtulu pielietot nevar. Piemēram,   f ( x ) := sin ⁡ x + 2 x {\displaystyle \ f(x):=\sin x+2x} un   g ( x ) := cos ⁡ x + 2 x {\displaystyle \ g(x):=\cos x+2x} . Tātad, argumentam x {\displaystyle x} tiecoties uz ∞ {\displaystyle \infty } , pastāv nenoteiktība ∞ ∞ {\displaystyle {\frac {\infty }{\infty }}} . Bet Lopitāla kārtulu pielietot nevar, jo lim x → ∞ f ′ ( x ) g ′ ( x ) = lim x → ∞ cos ⁡ x + 2 − sin ⁡ x + 2 {\displaystyle \lim _{x\to \infty }{\frac {f'(x)}{g'(x)}}=\lim _{x\to \infty }{\frac {\cos x+2}{-\sin x+2}}} neeksistē kosinusa un sinusa funkciju periodiskuma dēļ, argumentam x {\displaystyle x} tiecoties uz ∞ {\displaystyle \infty } . Tajā pat laikā sākotnējam dalījumam robežvērtība ir — to var atrast, pārveidojot sākotnējo dalījumu divos dalījumos: lim x → ∞ f ( x ) g ( x ) = lim x → ∞ sin ⁡ x + 2 x + cos ⁡ x − cos ⁡ x cos ⁡ x + 2 x = lim x → ∞ ( 1 + sin ⁡ x − cos ⁡ x cos ⁡ x + 2 x ) = 1 {\displaystyle \lim _{x\rightarrow \infty }{\frac {f(x)}{g(x)}}=\lim _{x\to \infty }{\frac {\sin x+2x+\cos x-\cos x}{\cos x+2x}}=\lim _{x\rightarrow \infty }\left(1+{\frac {\sin x-\cos x}{\cos x+2x}}\right)=1} .

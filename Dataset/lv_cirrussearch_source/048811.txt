Azerbaidžāna piedalījās 2000. gada vasaras olimpiskajās spēlēs, kas notika Austrālijas pilsētā Sidnejā no 2000. gada 15. septembra līdz 1. oktobra. Šī bija otrā reize, kad Azerbaidžāna piedalījās vasaras olimpiskajās spēlēs, kopš Padomju Savienības sabrukuma. Azerbaidžānas olimpisko komandu 2000. gada vasaras olimpiskajās spēlēs veidoja 31 sportisti, 25 vīrieši un 6 sievietes, kuri startēja 8 olimpiskajos sporta veidos. Visvairāk, 9 sportisti, no Azerbaidžānas startēja cīņas sportā. Azerbaidžāna šajās vasaras olimpiskajās spēlēs izcīnīja 3 olimpiskās medaļas (2 zelta un 1 bronzas medaļas), ierindojoties 61. vietā pēc iegūto medaļu kopskaita. Šīs bija pirmās vasaras olimpiskās spēles, kurās Azerbaidžāna izcīnīja zelta medaļas. Zelta medaļas izcīnīja sporta šāvēja Zemfira Meftahatdinova un cīņas sporta pārstāvis Namiks Abdullajevs, bet bronzas medaļu bokseris Vugars Alekperovs. Azerbaidžānas karognesējs olimpisko spēļu atklāšanas ceremonijā bija cīņas sporta pārstāvis Namiks Abdullajevs. Skatīt arī: 2000. gada vasaras olimpisko spēļu medaļu tabula un 2000. gada vasaras olimpisko spēļu medaļnieki Arifs Abdullajevs ( Cīņas sports) Namiks Abdullajevs ( Cīņas sports) Šamils Afandijevs ( Cīņas sports) Vugars Alekperovs ( Bokss) Elšads Allahverdijevs ( Cīņas sports) Magomeds Aripgadžijevs ( Bokss) Racebs Asabelijevs ( Cīņas sports) Mehmans Azizovs ( Džudo) Faigs Baghirovs ( Vieglatlētika) Hviča Bičinašvili ( Cīņas sports) Sergejs Bočkovs ( Vieglatlētika) Islams Dukučijevs ( Cīņas sports) Emīls Džabrajilovs ( Daiļlēkšana) Natiks Eivazovs ( Cīņas sports) Teimurs Gasimovs ( Vieglatlētika) Emins Gulijevs ( Peldēšana) Ruslans Hairovs ( Bokss) Ali Ismailovs ( Bokss) Elčins Ismailovs ( Džudo) Davuds Magomedovs ( Cīņas sports) Turans Mirzojevs ( Svarcelšana) Mahačs Nuridinovuns ( Bokss) Nizami Pašajevs ( Svarcelšana) Rasuls Salimovs ( Džudo) Elhans Suleimanovs ( Svarcelšana) Tarana Abasova ( Svarcelšana) Irada Ašumova ( Šaušana) Alisa Halejeva ( Peldēšana) Zulfija Huseinova ( Džudo) Aīda Isajeva ( Vieglatlētika) Zemfira Meftahatdinova ( Šaušana) «Azerbaijan at the 2000 Summer Games». sports-reference.com.

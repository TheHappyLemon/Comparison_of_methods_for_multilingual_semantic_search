Java (indonēziešu: Jawa) ir sala, Indonēzijas Republikas daļa, kurā mitinās aptuveni puse Indonēzijas iedzīvotāju, un tā dominē valstī politiski un ekonomiski. Salā atrodas Indonēzijas galvaspilsēta Džakarta. Starp citām pasaules salām Java atrodas 13. vietā pēc platības, kā arī arī 1. vietā pēc iedzīvotāju blīvuma — tajā dzīvo 140 miljoni iedzīvotāju (2020), kas ir vienlīdzīgi ap 1000 iedzīvotājiem uz km². Java atrodas uz austrumiem no Sumatras, uz rietumiem no Bali, uz dienvidiem no Borneo un uz ziemeļiem no Ziemsvētku Salas. Java atrodas seismiski aktīvā reģionā un izveidojusies galvenokārt vulkāniskās aktivitātes rezultātā, uz salas atrodas vismaz 38 kalni, kas kādreiz bijuši aktīvi vulkāni, arī salas augstākais punkts Semeru (3676 m) ir vulkāns. Pateicoties saviem kalniem un augstienēm, Java ir viens no rīsu audzēšanai piemērotākajiem reģioniem pasaulē.

nginx (angliskā izruna "engine X") ir viegls tīmekļa serveris / starpniekserveris un e-pasta (IMAP/POP3) starpniekserveris. 2016. gada janvārī nginx tika izmantots 141 443 630 domēnu apkalpošanai, tādējādi ieņemot 15,6% tirgus daļas, atpaliekot tikai no Apache un Microsoft pozīcijām. Nginx izejas kods tiek izplatīts ar BSD tipa licences nosacījumiem. Sākotnēji nginx tika izstrādāts Krievijā, lai spētu apkalpot dažādas augstas noslodzes tīmekļa lapas. Statisku failu servēšana, automātiska direktorija satura ģenerēšana Starpniekserveris (proxy) SSL atbalsts DNS un IP tipa virtuālie serveri FLV plūsmošana POP3 un IMAP proxy serveris STARTTLS atbalsts SSL stbalsts Nginx darbojas uz UNIX, Linux, OpenBSD, NetBSD, FreeBSD, Mac OS X un Solaris operētājsistēmām. «January 2016 Web Server Survey». netcraft.com.

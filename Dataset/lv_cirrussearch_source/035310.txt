jQuery ir starpplatformu JavaScript bibliotēka, kas izstrādāta, lai vienkāršotu HTML klientpuses skriptošanu. jQuery pašlaik ir izmantotākā JavaScript bibliotēka, tā ir uzstādīta 65% no 10 miljoniem visvairāk apmeklētāko mājaslapu. jQuery sintakse izstrādāta tā, lai būtu vieglāk pārvietoties dokumentā, atlasīt DOM elementus, veidot animācijas. jQuery ir atvērtā pirmkoda programmatūra, kas licencēta saskaņā ar MIT licenci. jQuery 3.1.0 Released – No More Silent Errors «License - jQuery Project». jQuery Foundation.

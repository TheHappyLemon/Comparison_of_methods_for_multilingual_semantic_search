Funkcija ir viena mainīgā atkarība no otra mainīgā, ja katrai neatkarīgā mainīgā vērtībai atbilst ne vairāk kā viena atkarīgā mainīgā vērtība. Funkcija ir definēta tad, ja norādīts piekārtojuma likums, pēc kura katrai argumenta vērtībai var atrast atbilstošo funkcijas vērtību. Piekārtojuma likumu var uzdot: ar vārdiem; ar tabulu; ar formulu; ar grafiku. Ja funkcionālo sakarību (atbilstību) starp diviem mainīgajiem x {\displaystyle x} un y {\displaystyle y} pieraksta šādi y = f ( x ) {\displaystyle y=f(x)} , tad y = f ( x ) {\displaystyle y=f(x)} , to sauc par funkcijas formulu, x {\displaystyle x} par neatkarīgo mainīgo jeb argumentu, bet y {\displaystyle y} par atkarīgo mainīgo jeb funkciju. Lasa: y {\displaystyle y} ir vienāds ar f {\displaystyle f} no x {\displaystyle x} . Pierakstā burts f {\displaystyle f} norāda likumu (kārtulu), pēc kura katrai argumenta x {\displaystyle x} vērtībai var noteikt atbilstošo y {\displaystyle y} vērtību. Gan funkciju, gan argumentu var apzīmēt arī ar citiem burtiem, piemēram, y = g ( x ) {\displaystyle y=g(x)} , y = h ( t ) {\displaystyle y=h(t)} , z = F ( y ) {\displaystyle z=F(y)} . Dažkārt funkciju apzīmē ar to pašu burtu, ar kuru apzīmēts atkarīgais mainīgais, piemēram, y = y ( x ) {\displaystyle y=y(x)} , s = s ( t ) {\displaystyle s=s(t)} , z = z ( y ) {\displaystyle z=z(y)} . Par funkcijas y = f ( x ) {\displaystyle y=f(x)} definīcijas apgabalu (pieļaujamo vērtību kopu) sauc visas tās neatkarīgā mainīgā x {\displaystyle x} vērtības, ar kurām izteiksmei f ( x ) {\displaystyle f(x)} ir jēga. Definīcijas apgabalu apzīmē ar simbolu D ( y ) {\displaystyle D(y)} vai D ( f ) {\displaystyle D(f)} . Saskaņā ar šo definīciju, funkcijas y = f ( x ) {\displaystyle y=f(x)} definīcijas apgabals D ( f ) {\displaystyle D(f)} sakrīt ar izteiksmes f ( x ) {\displaystyle f(x)} definīcijas apgabalu. Par funkcijas y = f ( x ) {\displaystyle y=f(x)} vērtību apgabalu sauc visas atkarīgā mainīgā y {\displaystyle y} vērtības. Vērtību apgabalu apzīmē ar E ( y ) {\displaystyle E(y)} vai E ( f ) {\displaystyle E(f)} . Funkciju sauc par pāra funkciju, ja katram x ∈ D {\displaystyle x\in D} izpildās vienādība f ( − x ) = f ( x ) {\displaystyle f(-x)=f(x)} . Funkciju sauc par nepāra funkciju, ja katram x ∈ D {\displaystyle x\in D} izpildās vienādība f ( − x ) = − f ( x ) {\displaystyle f(-x)=-f(x)} . Ja funkcijai neatbilst neviens no šiem abiem nosacījumiem, to sauc par ne pāra, ne nepāra funkciju. Funkciju sauc par periodisku, ja eksistē tāds pozitīvs skaitlis T {\displaystyle T} , ka katram x ∈ D {\displaystyle x\in D} pastāv sakarība f ( x + T ) = f ( x ) {\displaystyle f(x+T)=f(x)} . Mazāko no šādiem skaitļiem T {\displaystyle T} sauc par funkcijas periodu. Funkcija ir augoša, ja lielākai argumenta vērtībai atbilst lielāka funkcijas vērtība. Funkcija ir dilstoša, ja lielākai argumenta vērtībai atbilst mazāka funkcijas vērtība. Augošas un dilstošas funkcijas sauc par monotonām funkcijām. Katrai monotonai funkcijai pastāv apgrieztā (inversā) funkcija. Ja funkcijas y = f ( u ) {\displaystyle y=f(u)} arguments ir funkcija u = g ( x ) {\displaystyle u=g(x)} , funkciju y = f ( g ( x ) ) {\displaystyle y=f(g(x))} sauc par saliktu funkciju jeb funkciju f {\displaystyle f} (ārējā funkcija) un g {\displaystyle g} (iekšējā funkcija) kompozīciju. Vienas funkcijas ievietošanu citas funkcijas argumenta vietā sauc par superpozīciju. Lai uzzīmētu funkcijas y = f ( x ) {\displaystyle y=f(x)} grafiku, parasti sastāda vērtību tabulu, atbilstošos punktus ( x {\displaystyle x} , f ( x ) {\displaystyle f(x)} ) atliek koordinātu plaknē un caur šiem punktiem novelk nepārtrauktu līniju (vai līnijas). Ja koordinātu plaknē atliek punktus, kuru abscisa (x ass) ir funkcijas arguments (x vērtība), bet ordināta (Y ass) — atbilstošā funkcijas vērtība, tad visi šie punkti veido funkcijas grafiku. Elementārās pamatfunkcijas: pakāpes funkcija y = x α {\displaystyle y=x^{\alpha }} , eksponentā funkcija y = a x {\displaystyle y=a^{x}} , logaritmiskā funkcija y = log a ⁡ x {\displaystyle y=\log _{a}x} , trigonometriskās funkcija y = s i n x {\displaystyle y=sinx} , y = c o s x {\displaystyle y=cosx} , y = t g x {\displaystyle y=tgx} , y = c t g x {\displaystyle y=ctgx} , ciklometriskās funkcijas y = a r c s i n x {\displaystyle y=arcsinx} , y = a r c c o s x {\displaystyle y=arccosx} , y = a r c t g x {\displaystyle y=arctgx} , y = a r c c t g x {\displaystyle y=arcctgx} . Funkcijas, kuras iegūst no elementārajām pamatfunkcijām un konstantēm, galīgā skaitā izpildot ar tām saskaitīšanu, atņemšanu, reizināšanu, dalīšanu, saliktu funkciju veidošanas operāciju, sauc par elementārajām funkcijām. Tās iedala algebriskās un transcendentās funkcijās. Algebriskajām, aprēķinot vērtību, jāizpilda galīgs skaits algebrisku operāciju (saskaitīšana, atņemšana, reizināšana, dalīšana, kāpināšana racionāla skaitļa pakāpē), tās iedala racionālās — funkcijas izteiksme nesatur saknes — un iracionālās — funkciju izteiksmes satur saknes — funkcijās. Pārējās elementārās funkcijas ir transcendentās.

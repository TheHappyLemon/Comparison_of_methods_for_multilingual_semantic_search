Windows 3.1x (kodētais nosaukums Janus) ir 16 bitu operētājsistēmu virkne, kuru izmantoja uz personālajiem datoriem. Virkne sākās ar Windows 3.1, kurš pirmoreiz tika pārdots 1992. gada aprīlī kā Windows 3.0 turpinājums. Nākamās versijas tika publicētas starp 1992. un 1994. gadu, līdz brīdim, kad virkni pārtrauca Windows 95. Windows 3.1 tika izlaists 1992. gada 6. aprīlī, iekļāva TrueType fontu sistēmu. Windows 3.1 tika izveidots, lai nodrošinātu saderību ar vecākām versijām. Kā Windows 3.0 tas iekļāva File Manager un Program Manager. Windows 3.1 Multimedia PC versijā (tikai beta versija, izlaists 1992. gada novembrī) iekļāva multivides atskaņotāju, kas ļāva atskaņot video failus. Windows 3.11 tika izlaists 1993. gada 8. novembrī. Šis atjauninājums nepievienoja jaunas funkcijas, tikai izlaboja kļūdas. Microsoft nomainīja visas tirdzniecības Windows 3.1 versijas ar Windows 3.11, un piedāvāja bezmaksas atjauninājumus jebkuram Windows 3.1 īpašniekam. Windows 3.1 operētājsistēmai tika izstrādātas Internet Explorer 2, 3, 4, 5 versijas.

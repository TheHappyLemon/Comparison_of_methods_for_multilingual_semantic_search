Windows 1.0 ir Microsoft 16 bitu PC platformas operētājsistēma, kas izlaista 1985. gada 20. novembrī. Tā ir pirmā Microsoft operētājsistēma ar grafisko saskarni. Tās atbalsts beidzās 2001. gada 31. decembrī. Pirmā versija Windows 1.0 tika izlaista 1985. gada 20. novembrī. Windows 1.02, kas tika izlaists 1986. gada maijā, bija starptautiska versija. Windows 1.03, kas tika izlaists 1986. gada augustā, bija vairāk piemērots starptautiskai izdošanai, jo saturēja eiropiešu klaviatūru draiverus un citus papildus monitoru un printeru draiverus. Windows 1.04 tika izlaists 1987. gada aprīlī un bija saderīgs arī ar tikko izlaistajiem IBM PS/2 datoriem. 1987. gada novembrī Windows 1.0 aizstāja Windows 2.0. Microsoft atbalstīja Windows 1.0 16 gadus - līdz 2001. gada 31. decembrim. Tā ir visilgāk atbalstītā Microsoft Windows versija.

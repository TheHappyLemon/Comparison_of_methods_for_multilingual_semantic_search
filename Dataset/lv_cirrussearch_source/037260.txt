Windows NT ir operētājsistēmu saime, ko izstrādājis Microsoft. Tās pirmā versija tika izlaista 1993. gada jūlijā. Tā ir no procesora neatkarīga, vairākprocesu un vairāklietotāju operētājsistēma. Pirmā Windows NT versija bija Windows NT 3.1 un to radīja darbstacijām un serveriem. Tā bija paredzēta kā papildinājuma patērētājiem paredzētajām Windows versijām (ieskaitot Windows 1.0 līdz Windows 3.1x), kuras bija bāzētas uz MS-DOS operētājsistēmas. Pamazām Windows NT saime tika paplašināta Microsoft vispārīga lietojuma operētājsistēmu produktu līnijas virzienā visiem personālajiem datoriem, padarot Windows 9x operētājsistēmu saimi par novecojušu. Agrāk apzīmējums "NT" tika skaidrots ar frāzi "New Technology" (jaunā tehnoloģija), bet šādu nozīmi vairs neizmanto. Sākot ar Windows 2000 "NT" vairs netika izmantots produkta nosaukumā, un tika iekļauts tikai produkta versijas apzīmējumā. NT bija pirmā pilnībā 32 bitu Windows versija, kamēr patērētājiem orientētās versijas, Windows 3.1x un Windows 9x bija 16/32 bitu hibrīdversijas. Tā ir vairāku arhitektūru operētājsistēma. Sākotnēji tā atbalstīja vairākas centrālā procesora arhitektūras, ieskaitot IA-32, MIPS, DEC Alpha, PowerPC un vēlāk Itanium. Jaunākās versijas atbalsta x86 (konkrētāk IA-32 un x64) un ARM. Windows NT svarīgākās iespējas iekļauj Windows čaulu, Windows API, Native API, Aktīvo direktoriju, Group Policy, Hardware Abstraction Layer, NTFS, BitLocker, Windows Store, Windows Update un Hyper-V. Daļā Windows NT risinājumu priekšteči ir operētājsistēmas, ko Deivs Katlers izstrādāja, pirms tika pieņemts darbā Microsoft, konkrētāk VMS un RSX-11, kā arī neizlaistā objektos bāzētā operētājsistēma, ko viņš izstrādāja arhitektūrai DEC PRISM. Šo priekštecību Katlers skaidro Helēnas Kasteres grāmatas "Inside Windows NT" priekšvārdā.

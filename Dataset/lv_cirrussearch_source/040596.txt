Diferenciālvienādojums var būt homogēns kādā no diviem gadījumiem: 1) diferenciālā formā dota pirmās pakāpes diferenciālvienādojuma saskaitāmo koeficienti ir homogēnas mainīgo funkcijas; 2) jebkuras kārtas lineāra diferenciālvienādojuma gadījumā nav konstanta saskaitāmā. Vienkāršs pirmās pakāpes diferenciālvienādojums formā: M ( x , y ) d x + N ( x , y ) d y = 0 {\displaystyle M(x,y)\,dx+N(x,y)\,dy=0} ir homogēns, ja gan M(x, y), gan N(x, y) ir homogēnas funkcijas kopīgai pakāpei n. Jeb, reizinot katru mainīgo ar   λ {\displaystyle \lambda } , mēs iegūstam M ( λ x , λ y ) = λ n M ( x , y ) {\displaystyle M(\lambda x,\lambda y)=\lambda ^{n}M(x,y)\,} un N ( λ x , λ y ) = λ n N ( x , y ) . {\displaystyle N(\lambda x,\lambda y)=\lambda ^{n}N(x,y)\,.} Tātad M ( λ x , λ y ) N ( λ x , λ y ) = M ( x , y ) N ( x , y ) . {\displaystyle {\frac {M(\lambda x,\lambda y)}{N(\lambda x,\lambda y)}}={\frac {M(x,y)}{N(x,y)}}\,.} Attiecībā   M ( t x , t y ) N ( t x , t y ) = M ( x , y ) N ( x , y ) {\displaystyle {\frac {M(tx,ty)}{N(tx,ty)}}={\frac {M(x,y)}{N(x,y)}}} ,mēs varam ievietot t = 1 / x {\displaystyle t=1/x}   lai vienkāršotu šo attiecību par funkciju f {\displaystyle f} no viena mainīgā y / x {\displaystyle y/x} : M ( x , y ) N ( x , y ) = M ( t x , t y ) N ( t x , t y ) = M ( 1 , y / x ) N ( 1 , y / x ) = f ( y / x ) . {\displaystyle {\frac {M(x,y)}{N(x,y)}}={\frac {M(tx,ty)}{N(tx,ty)}}={\frac {M(1,y/x)}{N(1,y/x)}}=f(y/x)\,.} Ieviešam jaunu mainīgo y = u x {\displaystyle y=ux} ; to diferencējam, izmantojot reizināšanas likumu: d ( u x ) d x = x d u d x + u d x d x = x d u d x + u , {\displaystyle {\frac {d(ux)}{dx}}=x{\frac {du}{dx}}+u{\frac {dx}{dx}}=x{\frac {du}{dx}}+u,} Tādējādi pārveidojot doto diferenciālvienādojumu par diferenciālvienādojumu ar atdalāmiem mainīgajiem: x d u d x = f ( u ) − u ; {\displaystyle x{\frac {du}{dx}}=f(u)-u\,;} Šādā formā diferenciālvienādojums var tikt tieši integrēts un tādējādi atrisināts. Šī piemēra vienādības nevajag izmantot kā formulas atrisinājuma iegūšanai, tās ir šeit pieminētas tikai atrisinājuma gaitas demonstrēšanai. Pirmās pakāpes diferenciālvienādojums formā (a, b, c, e, f, g ir konstantes) ( a x + b y + c ) d x + ( e x + f y + g ) d y = 0 {\displaystyle (ax+by+c)dx+(ex+fy+g)dy=0\,} , kur af ≠ be, var tikt pārveidots par homogēnu, izmantojot lineāru mainīgo transformāciju abiem mainīgajiem ( α {\displaystyle \alpha } un β {\displaystyle \beta } ir konstantes): t = x + α ; z = y + β . {\displaystyle t=x+\alpha ;\,\,\,\,z=y+\beta \,.} Definīcija. Lineāru diferenciālvienādojumu sauc par homogēnu, ja izpildās sekojošs nosacījums: ja  ϕ ( x ) {\displaystyle \phi (x)}   ir atrisinājums, tad arī c ϕ ( x ) {\displaystyle c\phi (x)} ir atrisinājums, kur c {\displaystyle c} - nenulles konstante. Jāpievērš uzmanība tam, ka, lai izpildītos šis nosacījums, lineārā atkarīgā mainīgā y diferenciālvienādojumā katram saskaitāmajam jāsatur vai nu y, vai y atvasinājums. Lineārs diferenciālvienādojums, kuram šis nosacījums neizpildās, tiek saukts par nehomogēnu. Lineārs diferenciālvienādojums var būt pasniegts kā lineārs operators, kurš balstās uz y(x), kur parasti x ir neatkarīgais mainīgais un y - atkarīgais. Tātad vispārīgā veidā lineārs homogēns diferenciālvienādojums ir: L ( y ) = 0 {\displaystyle L(y)=0\,} kur L ir diferenciāloperators, atvasinājumu summa (definējot "0. atvasinājumu" kā sākotnējo, neatvasināto funkciju), katru reizinot ar funkciju   f i {\displaystyle f_{i}}   no x: L = ∑ i = 0 n f i ( x ) d i d x i , {\displaystyle L=\sum _{i=0}^{n}f_{i}(x){\frac {d^{i}}{dx^{i}}}\,,} kur   f i {\displaystyle f_{i}}   var būt konstantes, taču visi   f i {\displaystyle f_{i}}   vienlaicīgi nevar būt nulles. Piemēram, šis diferenciālvienādojums ir homogēns: sin ⁡ ( x ) d 2 y d x 2 + 4 d y d x + y = 0 , {\displaystyle \sin(x){\frac {d^{2}y}{dx^{2}}}+4{\frac {dy}{dx}}+y=0\,,} Taču šie divi ir nehomogēni: 2 x 2 d 2 y d x 2 + 4 x d y d x + y = cos ⁡ ( x ) ; {\displaystyle 2x^{2}{\frac {d^{2}y}{dx^{2}}}+4x{\frac {dy}{dx}}+y=\cos(x)\,;} 2 x 2 d 2 y d x 2 − 3 x d y d x + y = 2 . {\displaystyle 2x^{2}{\frac {d^{2}y}{dx^{2}}}-3x{\frac {dy}{dx}}+y=2\,.} Jāpiemin, ka konstanta saskaitāmā eksistence ir pietiekamais nosacījums, lai vienādība būtu nehomogēna, kā tas parādīts pēdējā piemērā.

Azerbaidžāna piedalījās 2004. gada vasaras olimpiskajās spēlēs, kas notika Grieķijas galvaspilsētā Atēnās no 2004. gada 13. līdz 29. augustam. Šī bija trešā reize, kad Azerbaidžāna piedalījās vasaras olimpiskajās spēlēs, kopš Padomju Savienības sabrukuma. Azerbaidžānas olimpisko komandu 2004. gada vasaras olimpiskajās spēlēs veidoja 36 sportisti, 30 vīrieši un 6 sievietes, kuri startēja 10 olimpiskajos sporta veidos. Atšķirībā no citiem gadiem, visvairāk sportistu, 9 sportisti, no Azerbaidžānas šajās vasaras olimpiskajās spēlēs bija pārstāvēti boksā. Azerbaidžāna pirmo reizi bija pārstāvēta tekvondo un vingrošanā. Azerbaidžāna šajās vasaras olimpiskajās spēlēs izcīnīja 5 olimpiskās medaļas (1 zelta un 4 sudraba medaļas), ierindojoties 50. vietā pēc iegūto medaļu kopskaita. No 5 medaļām pa divām medaļām tika izcīnītas boksā un šaušanā, bet viena medaļa tika izcīnīta cīņas sportā. Vienīgo zelta medaļu šajās olimpiskajās spēlēs izcīnīja cīkstonis Farids Mansurovs, savukārt bronzas medaļas izcīnīja sporta šāvēji Irada Ašumova un Zemfira Meftahatdinova, kā arī bokseri Fuads Aslanovs un Agasi Mammadovs. Azerbaidžānas karognesējs 2004. gada Atēnu spēļu atklāšanas ceremonijā bija svarcēlājs Nizami Pašajevs, savukārt noslēguma ceremonijā — cīkstonis Farids Mansurovs. Skatīt arī: 2004. gada vasaras olimpisko spēļu medaļu tabula un 2004. gada vasaras olimpisko spēļu medaļnieki Namigs Abdullajevs ( Cīņas sports) Džejhuns Abijevs ( Bokss) Rustams Agajevs ( Cīņas sports) Rašads Ahmadovs ( Tekvondo) Vugars Alakbarovs ( Bokss) Elmans Asgarovs ( Cīņas sports) Elnurs Aslanovs ( Cīņas sports) Vukars Aslanovs ( Cīņas sports) Fuads Aslanovs ( Bokss) Mehmans Azizovs ( Džudo) Sergejs Bočkovs ( Vieglatlētika) Sergejs Djačkovs ( Peldēšana) Teimurs Gasimovs ( Vieglatlētika) Ruslans Hairovs ( Bokss) Rovšans Huseinovs ( Bokss) Dadašs Ibrahimovs ( Vieglatlētika) Šahins Imranovs ( Bokss) Ali Ismajilovs ( Bokss) Elčins Ismajilovs ( Džudo) Asifs Malikovs ( Svarcelšana) Agasi Mammadovs ( Bokss) Farids Mansurovs ( Cīņas sports) Movluds Miralijevs ( Džudo) Turans Mirzajevs ( Svarcelšana) Alans Nanijevs ( Svarcelšana) Nijamadins Pašajevs ( Tekvondo) Nizami Pašajevs ( Svarcelšana) Vitālijs Rahimovs ( Cīņas sports) Alibajs Samadovs ( Svarcelšana) Alibajs Šukurovs ( Vieglatlētika) Džavīds Tagijevs ( Bokss) Irada Ašumova ( Šaušana) Jeļena Džemajeva ( Paukošana) Natālija Fiļina ( Peldēšana) Anna Gurbanova ( Vingrošana) Marina Lapina ( Vieglatlētika) Zemfira Meftahatdinova ( Šaušana) Azerbaijan at the 2004 Athina Summer Games

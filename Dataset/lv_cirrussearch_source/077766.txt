VIA OpenBook ir VIA Technologies ultraportatīvais dators ar brīvizmantošanai paredzētu un atvērtu specifikāciju modulāro platformu (reference design). Modulārās platformas specifikācijas kopš 2008. gada ir publiski pieejamas iespējamiem ražotājiem. Ultraportatīvo klēpjdatoru ražošanai paredzētā VIA OpenBook modulārās platformas specifikācija ir balstīta uz VIA C7, VIA C7-M vai VIA Nano procesora un integrētā VIA VX800 IGP mikroshēmojuma. Visa projekta informācija, ieskaitot izmantošanai gatavus dizaina failus CAD formātā, ir brīvpieejama un izmantojama licences ietvaros. No tirgzinības viedokļa projekts ir VIA Technologies sniegta iespēja ražotājam ātri un bez apgrūtinājumiem izstrādāt produktu, efektīvi samazinot adoptācijas laiku, izstrādes un ražošanas izmaksas, būtībā piedāvājot jau gatavu, piemērojamu produktu. 1.0+GHz VIA Nano-ULV procesors (spēj darboties 1.3Ghz takts frekvencē) VIA VX800 apvienotais mikroshēmojums DDR2 SO-DIMM operatīvā atmiņa līdz 2 GB apmēram 160GB cietais disks 8.9" WVGA 1024X600 LED ekrāns Grafiskais procesors: VIA Chrome9 HC3 DX9 3D dzinējs ar 256MB lielu atvēlētās operatīvās atmiņas apjomu Izmēri: 240 mm (platums) × 175 mm (garums) × 36.2 mm (augstums) (ar bateriju) Svars: mazāk par 1kg Tīkla karte: 10/100/1000 Mbit/s Broadcom Giga NIC Ethernet Solution Bezvadu interneta karte: Broadcom 802.11b/g vai GCT 802.16e vai Marvell sd8686 Wi-Fi, Bluetooth, WiMAX, EV-DO /W-CDMA, HSDPA, GPS - kā papildopcijas. "Četri vienā" iegultais karšu lasītājs 3 USB (Ver. 2.0 Type A Port) pieslēgvietas Audio: Realtek HD Audio codec, 2 skaļruņi 1 mikrofona pieslēgvieta 1 austiņu pieslēgvieta 2/0.3 megapikseļu divpusēja tīkla kamera 4 šūnu baterija VIA Technologies garantē pilnīgu savietojamību ar Windows XP un Windows Vista operētājsistēmām. VIA Technologies garantē savietojamību ar populārākajām Linux distribūcijām, lai gan daļai no aparatūras nemaz nav pieejami dziņi Linux sistēmām (piemēram, tīkla kamerai).

Windows Home Server ir Microsoft izstrādātā un 2007. gada 7. janvārī paziņotā servera operētājsistēma, kas ir paredzēta mājas lietošanai un tās galvenais uzdevums ir apvienot mājokļos esošos datorus, to datus, veikt datu dublēšanu un nodrošināt attālināto piekļuvi. Windows Home Server operētājsistēma ir bāzēta uz Windows Server 2003 SP2 sistēmas. Plašam patēriņam Windows Home Server tika nodots 2007. gada 16.jūlijā. Centralizēta datu dublēšana — veic datu dublēšanu līdz pat 10 datoriem, izmantojot Single instance technology tehniku, lai izvairītos no vairāku vienādu failu dublēšanas, neskatoties uz to, ka tie atrodas uz dažādiem datoriem. Datoru veselības uzraudzība — centralizēti tiek uzraudzīta visu tīklā esošo datoru tehniskais stāvoklis, kā arī antivīrusu un ugunsmūru stāvoklis. Datņu koplietošana — nodrošina datu glabāšanu uz servera un to lietošanu attālināti. Ir paredzētas speciālas kategorijas biežāk lietotajiem failiem kā mūzikai, video datiem, dokumentiem un citiem. Dati tiek indeksēti, lai nodrošinātu ātrāku meklēšanu. Printeru koplietošana — veic printera servera funkciju un vada drukāšanas darbus visiem lietotājiem. Iepriekšējās versijas — izmanto Volume Shadow Copy servisu, kas nodrošina datu dublēšanu konkrētos brīžos un ļauj atjaunot datus no iepriekš saglabātām versijām. Attālinātā administrēšana — nodrošina ar lietotāja saskarni, lai varētu veikt administratīvas darbības attālināti. Mediju straumēšana — var veikt mediju straumēšanu uz Xbox 360 vai citām ierīcēm, kas atbalsta Windows Media Connect. Datu dublēšana disku masīvos — nodrošina datu dublēšanu uz vairākiem cietajiem diskiem, lai izvairītos no to zaudēšanas, ja kāds no cietajiem diskiem atsakās. Uzlabošana ar papildinājumu palīdzību — servera darbība ir uzlabojama ar citu izstrādātāju lietotnēm un pielikumiem. Pielikumus var veidot uz Windows Home Server SDK. Windows Home Server balstās uz Microsoft Windows Server 2003 SP2 sistēmas, taču vadīšanas saskarne ir izstrādāta tāda, lai tā būtu lietotājam draudzīga un lai būtu iespējams vadīt serveri bez priekšzināšanām servera administrēšanā. Pie konfigurēšanas saskarnes, kura tiek saukta par Home Server Console, ir iespējams piekļūt no jebkura tīklā esošā datora kā arī no interneta. Ar Windows Home Server konsoles palīdzību ir iespēja veikt visas svarīgākās Home Server funkcijas, proti: Vadīt datu dublēšanas darbus - veidot dublēšanas darbus, uzraudzīt to statusu uz visiem datoriem, uz kuriem ir uzstādīts Windows Home Server Connector. Vadīt lietotāju kontus - ir iespēja izveidot līdz pat 10 lietotājiem, vadīt to piekļuvi pie koplietošanas mapēm, kā arī dot tiem iespēju veikt attālinātus pieslēgumus pie servera. Vadīt koplietošanas mapes - koplietošanas mapju sadaļa ir iespējams pievienot, noņemt un administrēt mapes ar . Vadīt servera glabātuves - kontrolēt pie servera pieslēgtos cietos diskus, uzraudzīt to stāvokli, atjaunot datus tajos, ja iespējams. Pielāgot servera funkcionalitāti - ir iespēja veikt dažas svarīgākās servera vadīšanas darbības kā atjaunināšanas darbības, servera datu dublēšanas, paroļu politikas, datumu un laiku, kā arī ir iespējams serveri pārstartēt. Sistēma piedāvā drošo SSL piekļuvi pie servera glabātajiem datiem caur internetu, izmantojot jebkuru interneta pārlūkprogrammu. Windows Home Server sola nodrošināt piekļuvi Web-saskarnei, izmantojot bezmaksas Windows Live nodrošināto adresi. Web-saskarne dod iespēju lejupielādēt un augšupielādēt datus no servera. Sistēma dod iespēju, izmantojot RDP vārteju, attālināti administrēt Windows Home Server tīklā esošos datorus. Šobrīd tas ir iespējams ar sistēmām, kas atbalsta Remote Desktop - tās ir Windows XP Professional, Tablet un Media Center un Windows Vista Business un Ultimate versijas.

Baltijas Sieviešu basketbola līga ir basketbola čempionāts, kurā piedalās spēcīgākās Baltijas valstu sieviešu basketbola komandas. Līga ir dibināta 1994. gadā. 1994—1995 : Telerina Vilnius 1995—1996 : Laisvė Kaunas 1996—1997 : Laisvė Kaunas 1997—1998 : Klondaika Rīga 1998—1999 : Horizont Minsk 1999—2000 : Lietuvos telekomas Vilnius 2000—2001 : Lietuvos telekomas Vilnius 2001—2002 : Lietuvos telekomas Vilnius 2002—2003 : Lietuvos telekomas Vilnius 2003—2004 : Lietuvos telekomas Vilnius 2004—2005 : Lietuvos telekomas Vilnius 2005—2006 : Lietuvos telekomas Vilnius 2006—2007 : Viļņas "TEO" 2007—2008 : Viļņas "TEO" 2008—2009 : Viļņas "TEO" 2009—2010 : Viļņas "TEO" 2010—2011 : Kauņas "VIČI-Aistės" 2011—2012 : Kauņas "VIČI-Aistės" 2012—2013 : Grodņas "Olimpija" 2013—2014 : Viļņas "Kibirkstis-Tiche" 2014—2015 : Minskas "Tsmoki-Minsk" BSBL mājas lapa Arhivēts 2009. gada 30. septembrī, Wayback Machine vietnē.

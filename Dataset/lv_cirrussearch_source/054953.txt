Oracle Database (bieži lietots nosaukums Oracle DB vai vienkārši Oracle), relāciju datubāzes pārvaldības sistēma (RDBPS), kuru izstrādā Oracle Corporation. Vēsturiski Oracle produktiem ir bijuši vairāki versiju numerācijas un nosaukumu veidošanas principi. Sākot ar Oracle 10g versiju, Oracle korporācija sāka standartizēt visu galveno produktu versijas, izmantojot 10g nosaukumu, tomēr daži avoti turpināja atsaukties uz Oracle Lietotņu versiju 11i. Sākot no 7. versijas, Oracle versiju numerācijā izmanto sekojošus kodus: Oracle7: 7.0.16 — 7.3.4 Oracle8 Database: 8.0.3 — 8.0.6 Oracle8i Database 1. laidiens : 8.1.5.0 — 8.1.5.1 Oracle8i Database 2. laidiens: 8.1.6.0 — 8.1.6.3 Oracle8i Database 3. laidiens: 8.1.7.0 — 8.1.7.4 Oracle9i Database 1. laidiens: 9.0.1.0 — 9.0.1.5 Oracle9i Database 2. laidiens: 9.2.0.1 — 9.2.0.8 Oracle Database 10g 1. laidiens: 10.1.0.2 — 10.1.0.5 Oracle Database 10g 2. laidiens: 10.2.0.1 — 10.2.0.3 Oracle Database 11g 1. laidiens: 11.1.0.6 — 11.1.0.7 Oracle Database 11g 2. laidiens: 11.2.0.1 Oracle Administratora Rokasgrāmata (Oracle Administrator's Guide) sniegta plašāka informācija par versiju numerāciju. Oracle korporācija piedāvā tabulu, kurā apkopoti pēdējās ielāpu kopas konkrētām versijām, operētājsistēmām un aparatūras arhitektūrām . Pirms Oracle 9i izlaišanas, Oracle korporācija pielāgoja datubāzes sistēmu plašam platformu spektram. Pēdējā laikā Oracle korporācija koncentrējusies uz mazāku operētājsistēmu un platformu spektru. Oracle 10g datubāzei ir sekojošu operētājsistēmu un platformu kombināciju atbalsts: Apple Mac OS X Server: PowerPC HP HP-UX: PA-RISC, Itanium HP Tru64 UNIX: Alpha HP OpenVMS: Alpha, Itanium IBM AIX5L: IBM POWER IBM z/OS: zSeries Linux: x86, x86-64, PowerPC, zSeries, Itanium Microsoft Windows: x86, x86-64, Itanium Sun Solaris: SPARC, x86, x86-64 Relāciju datubāzu tirgū Oracle Database sacenšas ar tādiem komerciāliem produktiem, kā IBM DB2 UDB un Microsoft SQL Server. Oracle un IBM sacenšas par datubāzu tirgu uz UNIX un Linux platformām, kamēr Microsoft dominē[nepieciešama atsauce] datubāzu tirgū uz Microsoft Windows platformām. «Oracle Database Patch Sets».

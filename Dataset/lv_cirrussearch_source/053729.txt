Kvadrātvienādojums ir otrās pakāpes vienādojums, kura vispārīgais veids ir a x 2 + b x + c = 0 , {\displaystyle ax^{2}+bx+c=0,\,} kur x {\displaystyle x} ir nezināmais un a {\displaystyle a} ≠ 0. Izteiksmi a x 2 + b x + c {\displaystyle ax^{2}+bx+c} sauc par kvadrāttrinomu. No algebras pamatteorēmas seko, ka kvadrātvienādojumam ir tieši divas saknes (šīs saknes var būt vienādas). Kvadrātvienādojuma a x 2 + b x + c = 0 {\displaystyle ax^{2}+bx+c=0} saknes x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} var aprēķināt pēc formulas x 1 , 2 = − b ± b 2 − 4 a c 2 a {\displaystyle x_{1,2}={\frac {-b\pm {\sqrt {b^{2}-4ac}}}{2a}}} jeb (izvērstā veidā) x 1 = − b + b 2 − 4 a c 2 a , x 2 = − b − b 2 − 4 a c 2 a . {\displaystyle x_{1}={\frac {-b+{\sqrt {b^{2}-4ac}}}{2a}},\quad \quad x_{2}={\frac {-b-{\sqrt {b^{2}-4ac}}}{2a}}.} Lietot var arī - x 1 , 2 = − b ± D 2 a {\displaystyle x_{1,2}={\frac {-b\pm {\sqrt {D}}}{2a}}} Lielumu D = b 2 − 4 a c {\displaystyle D=b^{2}-4ac\,} sauc par kvadrātvienādojuma a x 2 + b x + c = 0 {\displaystyle ax^{2}+bx+c=0} diskriminantu. Šis skaitlis nosaka kvadrātvienādojuma sakņu veidu: ja D > 0 {\displaystyle D>0} , tad kvadrātvienādojumam ir divas dažādas reālas saknes; ja D = 0 {\displaystyle D=0} , tad kvadrātvienādojumam ir divkārša sakne, kuru aprēķina pēc formulas x = − b / 2 a {\displaystyle x=-b/2a} ; ja D < 0 {\displaystyle D<0} , tad kvadrātvienādojumam ir divas kompleksi saistītas saknes. Ja ir zināmas kvadrātvienādojuma a x 2 + b x + c = 0 {\displaystyle ax^{2}+bx+c=0} saknes x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} , tad attiecīgo kvadrāttrinomu a x 2 + b x + c {\displaystyle ax^{2}+bx+c} var sadalīt reizinātājos: a x 2 + b x + c = a ( x − x 1 ) ( x − x 2 ) . {\displaystyle ax^{2}+bx+c=a(x-x_{1})(x-x_{2}).\,} Ja x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} ir kvadrātvienādojuma x 2 + p x + q = 0 {\displaystyle x^{2}+px+q=0} saknes, tad to summa ir vienāda ar koeficientu p {\displaystyle p} , kurš ņemts ar pretēju zīmi, bet sakņu reizinājums ir vienāds ar q {\displaystyle q} : x 1 + x 2 = − p , x 1 x 2 = q . {\displaystyle x_{1}+x_{2}=-p,\quad \quad x_{1}x_{2}=q.} Vispārīgā gadījumā, ja kvadrātvienādojums ir formā a x 2 + b x + c = 0 {\displaystyle ax^{2}+bx+c=0} , kur a {\displaystyle a} ≠ 0, un x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} ir tā saknes, tad x 1 + x 2 = − b / a , x 1 x 2 = c / a . {\displaystyle x_{1}+x_{2}=-b/a,\quad \quad x_{1}x_{2}=c/a.} Šo apgalvojumu sauc par Vjeta teorēmu, jo to pirmais pierādīja franču matemātiķis Fransuā Vjets. Vjeta teorēmas pierādījums Ja x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} ir kvadrātvienādojuma x 2 + p x + q = 0 {\displaystyle x^{2}+px+q=0} saknes, tad ir spēkā vienādība ( x − x 1 ) ( x − x 2 ) = x 2 + p x + q . {\displaystyle (x-x_{1})(x-x_{2})=x^{2}+px+q.\,} Atverot iekavas, iegūstam ( x − x 1 ) ( x − x 2 ) = x 2 − x 1 x − x 2 x + x 1 x 2 = x 2 − ( x 1 + x 2 ) ⏟ p x + x 1 x 2 ⏟ q . {\displaystyle {\begin{aligned}(x-x_{1})(x-x_{2})&=x^{2}-x_{1}x-x_{2}x+x_{1}x_{2}\\&=x^{2}\underbrace {-(x_{1}+x_{2})} _{p}x+\underbrace {x_{1}x_{2}} _{q}.\end{aligned}}} Vispārīgā gadījumā, ja kvadrātvienādojums ir formā a x 2 + b x + c = 0 {\displaystyle ax^{2}+bx+c=0} , kur a {\displaystyle a} ≠ 0, tad abas vienādojuma puses var izdalīt ar a {\displaystyle a} un iegūt kvadrātvienādojumu x 2 + b a x + c a = 0. {\displaystyle x^{2}+{\frac {b}{a}}x+{\frac {c}{a}}=0.} Šis kvadrātvienādojums ir formā x 2 + p x + q = 0 , {\displaystyle x^{2}+px+q=0,} kur p = b / a {\displaystyle p=b/a} un q = c / a {\displaystyle q=c/a} , tāpēc tam var pielietot augstāk aprakstīto vienkāršoto Vjeta teorēmas versiju. Rezultātā iegūst x 1 + x 2 = − b / a , x 1 x 2 = c / a , {\displaystyle x_{1}+x_{2}=-b/a,\quad \quad x_{1}x_{2}=c/a,} kur x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} ir vienādojuma x 2 + p x + q = 0 {\displaystyle x^{2}+px+q=0} saknes. Tā kā šis vienādojums ir ekvivalents sākotnējajam vienādojumam, tad x 1 {\displaystyle x_{1}} un x 2 {\displaystyle x_{2}} ir arī vienādojuma a x 2 + b x + c = 0 {\displaystyle ax^{2}+bx+c=0} saknes.

Elektriskā lādiņa nezūdamības likums: no apkārtējiem ķermeņiem izolētas sistēmas kopējais elektriskais lādiņš laikā nemainās (to nevar patvaļīgi radīt vai iznīcināt). Ja sistēma sastāv no vairākiem ķermeņiem, kuru kopējais elektriskais lādiņš q = q 1 + q 2 + . . . + q n   {\displaystyle q=q_{1}+q_{2}+...+q_{n}\ } , tad no elektriskā lādiņa nezūdamības likuma izriet, ka q = c o n s t   {\displaystyle q=const\ } jeb d q d t = 0   {\displaystyle {\frac {\mathrm {d} q}{\mathrm {d} t}}=0\ } . Elektriskie lādiņi ķermeņu mijiedarbībā var pārkārtoties, bet tikai tā, lai saglabātos sistēmas kopējais lādiņš q   {\displaystyle q\ } . Lādiņa nezūdamības likums ir universāls likums, un dabā nav sastopami izņēmumi. Tas ir spēkā visās elementārdaļiņu reakcijās, kā arī ķermeņu elektriskās uzlādēšanās un polarizēšanās procesos. Gadījumā, kad sistēma nav izolēta (slēgta), elektriskais lādiņš no tās var aizplūst vai arī tajā ieplūst. Iedomājoties, ka elektriski lādētu ķermeņu vai daļiņu kopu aptver noslēgta virsma S   {\displaystyle S\ } , caur kuru var pastāvēt lādiņa q   {\displaystyle q\ } plūsma. Tādā gadījumā lādiņu nezūdamības likumu raksta šādi: − d q d t = I   {\displaystyle -{\frac {\mathrm {d} q}{\mathrm {d} t}}=I\ } , kur I   {\displaystyle I\ } - elektriskā strāva, kura šķērso slēgto virsmu S   {\displaystyle S\ } (A) d q d t   {\displaystyle {\frac {\mathrm {d} q}{\mathrm {d} t}}\ } - sistēmas summārā lādiņa izmaiņas ātrums (C/s) Mīnusa zīme nozīmē to, ka ir pieņemts, ka, samazinoties pozitīvajam lādiņam q   {\displaystyle q\ } , lādiņnesēji aizplūst no virsmas S   {\displaystyle S\ } ierobežotā tilpuma.

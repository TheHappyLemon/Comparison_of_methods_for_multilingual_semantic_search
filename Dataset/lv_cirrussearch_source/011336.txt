Adaptive Server Enterprise ir datu bāzu pārvaldības sistēma uzņēmumiem un organizācijām. Ekstrēmām transakciju pārstrādes sistēmām jāspēj atbalstīt desmitiem tūkstošu vienlaicīgu lietotāju ar ļoti augstām prasībām un tajā pašā laikā arī pēc iespējas zemākām izmaksām. ASE dara visu to un vēl vairāk. Darbinot 46 no top 50 globālajām bankām un drošības firmām, ASE uzņēmumu datubāzu pārvaldes sistēma ir pazīstama ar tās lielo uzticamību, ļoti augsto veiktspēju un zemajām uzturēšanas izmaksām. Piedāvā ļoti augstu veiktspēju: Ļauj prasmīgām IT aparatūrām apstrādāt miljoniem transakciju minūtē ar terabaitiem lielām datubāzēm, tajā pašā laikā nodrošinot straujas izaugsmes tempus datu un transakciju ilgumā. Samazina riskus: Ļauj uzņēmumiem izvairīties no dārgiem un nepatīkamiem traucējumiem, kā piemēram, sistēmu sabrukumiem, neparedzētām nelaimēm vai arī datu zudumiem. Sybase’s Linux sadarbība sākās 1999. gadā ar biznesa klases programmas DBMS izveidi uz Linux un bija pirmais, kas piedāvāja bezmaksas biznesa klases tirdzniecības datubāzes, kas iekļāva izmēģinājuma attīstību un izvietošanu. Izmantojot ASE augsto uzticamību, ASE un Linux piedāvā ļoti augstu uzticamības līmeni, zemus izmaksu risinājumus, kas var pielāgoties individuālajām servisa vajadzībām katrai lietotnei un pielāgot visu jaunajām prasībām. ASE Options Guide: Interaktīvā tiešsaistes brošūra izceļ plašās iespējas kā palielināt ASE jaudu, lai tā atbilstu jūsu nepieciešamajai veiktspējai, pieejamībai, efektivitātei un drošībai. ASE In-Memory Databases Options: Pilnībā integrēti iekš ASE, In-Memory Databases Options apstrādā lietotnes ar tiešu atbildi un ļoti lielu caurlaidību. Iekšējās atmiņas datubāzes atrodas tikai atmiņā, līdz ar to diskus tas neizmanto. ASE Advanced Backup Services – TivoliTM Storage Manager Option: Atbalsta Adaptive Server Backup Server integrēšanu ar Tivoli Storage Manager. ASE Partition Option: ļauj lietotājam pārvaldīt lielas tabulas un indeksus, sadalot tos mazākos, pārskatāmākos un vieglāk pārvaldāmos gabalos jeb partīcijās. ASE Encrypted Columd Option: Ļauj datus šifrēt un noglabāt tos ar ASE pēc savām vajadzībām. ASE Security & Directory Services Package: Nodrošina datu privātumu ar rindu bāzu pieejas kontroli, iekšējo datu šifrēšanu un atbalstu LDAP, Active Directory un Pluggable Authentification Modules (PAM) iespējām. ASE Active Messagign Option: Piedāvā veidu, kā notvert transakcijas Adaptive Server datubāzē un nogādāt tās kā notikumus ārējām lietotnēm izmantojot kādu no šiem: JMS ziņu pāreju, EAServer un Sonic Software; vai Message Queue Interface (MQI) ar WebSphere MQ.

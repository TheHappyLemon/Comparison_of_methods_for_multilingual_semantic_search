Microsoft SharePoint ir programmatūras produktu un komponentu kolekcija, kas ietver šādus produktus un sastāvdaļas: SharePoint Foundation; SharePoint Server; SharePoint Online; SharePoint Designer; SharePoint Workspace. SharePoint var izmantot, lai veidotu vietnes, kas sniedz lietotājiem iespēju strādāt patstāvīgi. Vietnes, kuras tika veidotas uz SharePoint platformas var tikt izmantotas kā informācijas, zināšanu un dokumentu krātuve, kā arī tiek izmantots, lai palīdzētu veikt mijiedarbību tīmekļa lietojumprogrammās, piemēram, wiki un blogus. Lietotāji var vadīt un mijiedarboties ar sarakstiem un dokumentiem bibliotēkās, izmantojot kontroles informāciju, saucamās par Web daļas(SharePoint WebParts) SharePoint Foundation. Bezmaksas Windows Server lietotne. Pamattehnoloģija, kas izmantota visās SharePoint vietnēs. SharePoint Foundation ir pieejama izvietošanai lokāli bez maksas, un iepriekšējās versijās to sauca Windows SharePoint Services. Varat izmantot SharePoint Foundation, lai ātri izveidotu dažādu tipu vietnes un sadarbotos, strādājot ar tīmekļa lapām, dokumentiem, sarakstiem, kalendāriem un datiem. SharePoint Server. Maksas SharePoint pakalpojums, lai veiktu MS Office pieteikumu funkcionalitātes integrāciju. Tā ir veidota uz Microsoft SharePoint Foundation un paplašina savas spējas. Microsoft Project Server šobrīd ir jauns papildinājums, kas tiek uzstādītas uz Microsoft SharePoint Server 2010 bāzes. SharePoint Server nodrošina visus SharePoint Foundation līdzekļus, kā arī papildlīdzekļus un iespējas, piemēram, uzņēmuma informācijas satura pārvaldību, biznesa informāciju, uzņēmumiem paredzētu meklēšanu un personiskus profilus, izmantojot līdzekli Manas vietnes. SharePoint Server ir pieejams izvietošanai lokāli mākoņpakalpojuma piedāvājuma, piemēram, Microsoft Office ietvaros. Microsoft Office SharePoint Designer (SPD) - HTML redaktors WYSIWYG (iepriekš - FrontPage) stilā, tika optimizēts, lai radītu SharePoint lapas un dokumentu pārvaldību WSS vietnēm. SPD nodrošina piekļuvi apmetuma dzinēju Microsoft Expression Web funkcionalitātei caur Microsoft Visual Studio izstrādes vidi. Kopš 2009 gada pavasara kļuva bezmaksas produkts.

Movable Type is a weblog publishing system developed by the company Six Apart. It was publicly announced on September 3, 2001; version 1.0 was publicly released on October 8, 2001. The current version is 8.0. Movable Type is proprietary software. From June 2007 to July 2013, Six Apart ran the Movable Type Open Source Project, which offered a version of Movable Type under the GPL.

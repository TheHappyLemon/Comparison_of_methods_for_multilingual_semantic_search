Here is wikipedia dump files stored, but only locally, beccause github allows only to store files less than 1GB for free and paid plan is 5$/month. Therefore I removed this file from repo history, if you clone this repo make sure to add neded wikipedia dumps here.

1) 'lvwiki-20240401-pages-articles-multistream.xml' - regular dump from https://dumps.wikimedia.org/ -> https://dumps.wikimedia.org/backup-index.html -> https://dumps.wikimedia.org/lvwiki/20240401/ -> 
2) 'lvwiki-20240401-cirrussearch-content.json' - clear dump from https://dumps.wikimedia.org -> https://dumps.wikimedia.org/other/ ->
https://dumps.wikimedia.org/other/cirrussearch/ -> 202401 -> lvwiki-20240401-cirrussearch-content.json which is cleared text of each wikipedia page in format needed for cirrussearch extension, which is based on elasticsearch engine.
3) 'enwiki-20240401-cirrussearch-content' - clear dump from https://dumps.wikimedia.org/other/cirrussearch/20240401/ -> enwiki-20240401-cirrussearch-content.json.gz
